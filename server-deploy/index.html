Requirements¶

  • Docker

Server start¶

  • Download the buffy docker-compose file 
wget -O docker-compose.yaml https://git.connect.dzd-ev.de/dzdpythonmodules/buffy/-/raw/main/docker-compose.yaml?inline=false
  • Start the Buffy server with docker compose
docker-compose up -d
